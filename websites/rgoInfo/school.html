The Microsoft Software and Systems Academy (MSSA) is an 18 week program with prerequisites hosted by Embry-Riddle Aeronautical University designed to prepare transitioning service members and veterans for a career in the information technology field.

The Cloud Applications Developer Specialization teaches you to design, build, and deploy applications to the web using cloud-based services. You'll start by acquiring programming skills with C# and data management skills with SQL, then apply those capabilities to build web applications with Microsoft's ASP.NET MVC framework.

You'll dive deep into what it takes to construct and maintain modern "web-apps" and culminate by deploying your own application to the cloud. Throughout, you'll learn the conceptual frameworks and gain the hands-on skills you need to master cloud-centric backend web development.
